# Regras Gerais do Nexus FA

Este arquivo define as diretrizes de desenvolvimento para o projeto Nexus FA.

- **Design**: Utilizar o estilo "Nexus Operational" com verde neon (#22C55E) e fundo escuro (#0B0B0B).
- **Tradução**: Manter todos os termos da interface em Português do Brasil (ex: Operacional, Cargos, Etapas).
- **Backend**: Utilizar Supabase com as tabelas `fa_roles`, `fa_pipeline_stages` e `fa_pipeline_tasks`.
- **Animações**: Utilizar `framer-motion` para transições suaves.
